# Praticando Java — Formação Alura

Este repositório faz parte da formação **Praticando Java** da Alura e reúne uma coleção de pacotes contendo exercícios, desafios e pequenos projetos desenvolvidos ao longo dos cursos. Cada *package* segue o padrão `br.com.nadson.desafioscX`, onde **"X" representa o curso da formação**. Dentro de cada pacote, há desafios que praticam os conceitos apresentados na trilha.

## Objetivo do Repositório

Consolidar o aprendizado
