# Desafios C7 --- Regras de Negócio e Modelagem de Objetos

Este módulo contém mini-projetos com regras de negócio mais definidas,
incluindo estoques e controle de empréstimos.

## Conteúdos abordados

-   Modelagem de classes com atributos privados
-   Regras de negócio: estoque, vendas, datas de entrega
-   Uso de `LocalDate`
-   Estruturação orientada a objetos

## Exemplos de desafios

### Controle de Estoque

-   Registrar produto
-   Validar vendas
-   Impedir vendas acima do estoque

### Empréstimo de Livro

-   Verificar atraso
-   Calcular multa
-   Validar datas
