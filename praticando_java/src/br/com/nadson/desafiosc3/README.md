# Praticando Java: Laços \*for\* e \*while\*



Este módulo corresponde ao \*\*Curso 3 da formação “Praticando Java” da Alura\*\*.

Aqui estão os exercícios práticos destinados ao domínio dos principais laços de repetição no Java, aplicando lógica estruturada e controle de fluxo.



\## Conteúdo estudado



\* Uso do laço `while`

\* Uso do laço `for`

\* Controle de variáveis de iteração

\* Incrementos e decrementos

\* Laços condicionais e interrupções (`break` e `continue`)

\* Construção de loops com condições variadas

\* Processamento iterativo de listas e sequências numéricas



\## Estrutura do pacote



```

desafiosc3/

```



\## Como executar



1\. Abra o projeto em sua IDE (Eclipse, IntelliJ IDEA ou VS Code).

2\. Navegue até:



&nbsp;  ```

&nbsp;  praticando\_java/src/br/com/nadson/desafiosc3/principal/

&nbsp;  ```

3\. Execute a classe que contém o método `public static void main`.



\## Objetivo do módulo



Aprimorar sua capacidade de repetir operações de forma eficiente, controlar fluxos de execução e manipular coleções simples usando \*loops\*. Esse conhecimento será essencial para módulos posteriores como Strings, coleções e orientação a objetos.
