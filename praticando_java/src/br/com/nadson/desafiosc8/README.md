# Desafios C8 --- Sistemas Simples com Lógica de Negócio

Este pacote possui sistemas independentes simulando cenários reais como
controle bancário, bateria, avaliação de filmes etc.

## Conteúdos abordados

-   Classes orientadas a objetos com responsabilidades claras
-   Regras para depósito e saque
-   Enumerações (níveis de bateria)
-   Validação de notas e cálculo de média

## Exemplos de mini-sistemas

### Controle de Bateria

-   Carregamento
-   Descarga
-   Limites e status baseado no nível

### Avaliação de Filmes

-   Notas de 0 a 5
-   Média automática

### Conta Bancária

-   Saque, depósito e validações
