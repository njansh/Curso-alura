# Desafios C6 --- Manipulação de Coleções e Streams

Os desafios deste pacote exploram `List`, `ArrayList` e operações com
**Java Streams**, incluindo soma, filtragem e transformações.

## Conteúdos abordados

-   Listas e coleções
-   Métodos como `remove`, `add`, `forEach`
-   Redução e soma com `stream()`
-   Operações funcionais

## Objetivo

Fixar o uso de coleções Java e introduzir o paradigma funcional aplicado
ao processamento de listas.
