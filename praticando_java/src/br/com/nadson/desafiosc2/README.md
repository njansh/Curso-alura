\## Praticando Java: condicionais if e else



Este módulo contém exercícios práticos desenvolvidos durante o curso \*\*Praticando Java: condicionais if e else\*\*, cujo objetivo é reforçar a lógica condicional em Java por meio de situações reais, decisões baseadas em valores e validações.



---



\## Objetivos do módulo



\* Utilizar \*\*estruturas condicionais\*\* (if, else if, else).

\* Comparar números, textos e expressões booleanas.

\* Criar fluxos de decisão para diferentes cenários.

\* Desenvolver pequenos programas Java focados em tomada de decisão.



---



\## Conteúdos praticados



\* Operadores relacionais e lógicos

\* Comparações numéricas e textuais

\* Estruturas condicionais simples e compostas

\* Blocos e escopo

\* Mensagens e fluxos dependendo da entrada do usuário



---



\## Tecnologias utilizadas



\* \*\*Java 17+\*\*

\* \*\*JDK\*\*

\* \*\*IDE\*\* (Eclipse, IntelliJ ou VS Code)

\* \*\*Console\*\* para entrada e saída padrão



---



\## Como executar



1\. Certifique-se que o Java está instalado:



&nbsp;  ```

&nbsp;  java -version

&nbsp;  ```

2\. No terminal, navegue até a pasta do módulo:



&nbsp;  ```

&nbsp;  cd src/br/com/nadson/desafiosc2/principal

&nbsp;  ```

3\. Compile o programa:



&nbsp;  ```

&nbsp;  javac NomeDaClasse.java

&nbsp;  ```

4\. Execute:



&nbsp;  ```

&nbsp;  java NomeDaClasse

&nbsp;  ```
